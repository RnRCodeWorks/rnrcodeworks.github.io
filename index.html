Project: Cryptid Finder

Overview:

Cryptid is a 3-5 player board game in which each player is trying to be the first to figure out where the cryptid locations is. Each player is given a different valid location rule, i.e "Within two of a swamp", based on the board layout and player count. Each turn a player is allowed to ask one other player if a location is valid based on that players rule.

The idea behind this project was just code practice and to see if any maps configurations had multiple right answers. Currently it does not account for advanced maps, they contain one more color of building and inverse clues. Given the board layout code, ie 453261, and player count, it will know the orientation of each board, structure locations, associated clues.
